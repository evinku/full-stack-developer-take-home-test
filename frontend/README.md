# Frontend (Next.js)

## Quick Start

### Environment Setup
```
# Copy .env.local.example to .env.local
cp .env.local.example .env.local
```

###  Start the Local Frontend (Running on http://localhost:3000)
```
# Install dependencies and start the frontend
npm install
npm run dev
```

## Testing

```
# Execute tests
npm run test
```
